    CONSOLE APPLICATION : TeamDungeon_CIS150_FinalProject Project Overview
========================================================================

To play the game Dungeons and Goblins, you will first type in your name, then press enter.

After the greeting screen, the game will start. You nagivate the game through a main menu with options that include
moving the character, checking health, checking inventory, and admin controls.

To choose your option, you will enter the integer listed. Most selections and options will involve entering in a
corresponding integer.

Some issues we had with the code was connecting different sections of code all together, initializing and mastering
GitHub, simple mistakes and typos, and clashes.

Some things we were unable to complete were dead ends, sounds, and more graphics
